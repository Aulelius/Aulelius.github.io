1. Tripinsite

여행객들이 자신이 원하는 관광, 맛집기행과 같은 테마에 맞춰 다른사람들의 여행일정을 검색하고 한국관광공사에서 제공하는 데이터를 활용하여 정확한 관광지 정보를 얻게 하였습니다.

프로젝트 기간 : 2019년 5월 17일 ~ 2019년 7월 19일

운영환경 : Apache Tomcat 8.5

프론트엔드 : CSS, JS, HTML, Bootstrap, jQuery, Ajax

백엔드 : JSP, Java, mybatis, MySQL

프레임워크 : Spring Framework

사용툴 : Eclipse IDE

사용API : 국내 관광정보 서비스 API, 다음 맵 API
